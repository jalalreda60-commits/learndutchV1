# App Images

| File | Size | Purpose |
|------|------|---------|
| icon.png | 1024×1024 | App icon |
| splash.png | 1284×2778 | Splash screen |
| adaptive-icon.png | 1024×1024 | Android adaptive icon |
| favicon.png | 48×48 | Web favicon |
| notification-icon.png | 96×96 | Push notification icon |
